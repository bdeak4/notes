# DNS tools

web tools:

- [DNSTools](https://dnstools.ws/) - lookup, ping, traceroute and other utilities
- [DNSdumpster](https://dnsdumpster.com/) - dns recon & research, find & lookup dns records
- [nip.io](https://nip.io/) - wildcard DNS for any IP Address

cli dns lookup:

```
$ host example.com
```
